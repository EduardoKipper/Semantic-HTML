HTML

O HTML como seu próprio nome diz, é uma linguagem de marcação de hipertexto, onde utilizamos tags para separar e organizar textos, imagens, vídeos, etc. de forma que o navegador do usuário consiga ler e apresentar o que fora passado para uma página web.

Tags

As tags do HTML são elementos fundamentais que estruturam e formatam o conteúdo de uma página da web. São compostas por palavras-chave ou códigos inseridos entre os símbolos de "Maior e Menor" (< >), delimitando elementos específicos dentro de um documento HTML. As tags desempenham papéis distintos, permitindo a marcação de diferentes tipos de conteúdo, como texto, imagens, links e elementos multimídia. Além disso, as tags podem conter atributos, que fornecem informações adicionais sobre o elemento, como sua aparência visual, comportamento ou relação com outros elementos. Por meio da utilização adequada e combinada de tags, desenvolvedores web podem criar páginas estruturadas e semânticas, facilitando a interpretação e processamento por parte dos navegadores e motores de busca.

Estrutura Mínima

A estrutura mínima de uma página em HTML 5 é a seguinte:

DocType

Estrutura Mínima

                                
    <!DOCTYPE html>
        <html lang="pt-br">
            <head>
                <title>Seu primeiro HTML</title>
                <meta charset="utf-8">
            </head>
            <body>
                <p>O que vai aparecer na sua página</p>
            </body>
        </html>

  3. O atributo lang= “pt-br” que posto como atributo ao <html> determina a língua utilizada na página;

Tags de Texto

São a base principal de uma página, ou seja, o conteúdo dela.

Para títulos podemos utilizar as tags <h1> - <h6> que em ordem crescente vão diminuindo o tamanho dos títulos. Uma página deve conter pelo menos uma tag <h1> (princípio de SEO) Para parágrafos podemos utilizar a tag <p>;

No HTML temos os chamados Elementos em Bloco e Elementos em Linha, resumidamente, Elementos em Bloco ocupam a largura total da página e os Elementos em Linha que podemser postos tanto um ao lado do outro como até mesmo dentro de alguns Elementos em Bloco.

Aqui alguns exemplos de Elementos em Bloco:
  • <div> (division, ou divisão, tag bloco genérica)
  • <h1> até <h6> (títulos)
  • <p> (parágrafo)
  • <blockquote> (citação em bloco)
  • <b> (põe o bloco em negrito)
  • <i> (põe o bloco em itálico)
  • <ul> (lista não ordenada)
  • <ol> (lista ordenada)
  • <form> (para formulários)
  • etc.
E aqui alguns exemplos de Elemento em Linha:
  • <span> (tag em linha genérica)
  • <strong> (põe a linha em negrito)
  • <em> (põe a linha em itálico)
  • <sup> (SuperScript A2)
  • <sub> (SubScript B8)
  • <a> (âncora, usada para links)
  • <img> (imagem)
  • etc.

Tags Âncoras

As âncoras são como uma ponte entre um site ou outro, no html a tag <a> é utilizada para aplicar tais "links" no texto.

Dentro da tag <a> utilizamos o atributo href para indicar a referência do hiperlink.

E depois acrescentamos o conteúdo do link ou o "nome explicativo" dele.

Como por exemplo no código abaixo:

<a href="Referencia.html">Seu Link</a>

Que gera o seguinte resultado: Seu Link
