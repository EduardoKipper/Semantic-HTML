Elementos

O que são?

Os elementos em HTML nada mais são que conjuntos de tags que formam a estrutura da página, podendo eles também assumirem diversos atributos, como classes, id's, entre muitos outros.

Logo abaixo veremos três dos principais elementos do HTML:

  • <head> - é um elemento onde se encontram todas as informações da página.
  • <body> - tudo que estiver dentro do elemento será exibido no navegador dentro da página.
  • <script> - elemento com função de portar scripts (em javascript) dentro do html.

Semânticos:

Também existem os chamados elementos semânticos do HTML, que não possuem estilos próprios nem modificam o código em si porém tem como finalidade organizar o código, facilitando tanto para os motores de busca da internet, como para a manutenção da página.

Alguns dos elementos semanticos são:

  • <header> - Define o cabeçalho da página ou de uma seção específica.
  • <nav> - Indica uma seção de navegação, geralmente contendo links para outras partes do site.
  • <main> - Define o conteúdo principal da página, excluindo cabeçalho, rodapé e outras áreas de navegação.
  • <article> - Representa um conteúdo independente e autossuficiente, como um post de blog ou uma notícia.
  • <section> - Define uma seção genérica em um documento, frequentemente utilizada para agrupar conteúdo relacionado.
  • <aside> - Representa conteúdo que está relacionado ao conteúdo circundante, mas não é sua parte principal.
  • <footer>: Define o rodapé da página ou de uma seção específica, geralmente contendo informações como direitos autorais, informações de contato, etc.
